## 📋 О проекте

AstroMeet — это мини-приложение ВКонтакте, которое использует ведическую астрологическую систему для анализа совместимости между мужчиной и женщиной. Приложение основано на 108-падовой системе (27 накшатр × 4 пады) и предоставляет подробный анализ по 8 параметрам совместимости (Кута).

### 🔍 Ключевые особенности

- **108-падовый анализ**: Уникальная система, учитывающая 108 комбинаций пад для мужчин и женщин
- **8 параметров совместимости**: Полный расчёт по Варне, Вашье, Йони, Гане, Нади, Маитри, Стха-Куте и Бхакут
- **Интерактивный интерфейс**: Визуальные барабаны для выбора пад, цветовая индикация результатов
- **Подробные рекомендации**: Персонализированные советы на основе результатов анализа
- **Адаптивный дизайн**: Полная поддержка мобильных устройств и интеграция с VK Mini Apps

## 🚀 Функциональность

### Основные возможности

1. **Выбор пад**
   - Интерактивные барабаны для выбора мужской и женской пад
   - Автоматическое определение накшатры по выбранной паде
   - Визуальное отображение характеристик выбранных пад

2. **Расчёт совместимости**
   - Мгновенный расчёт процента совместимости (0-100%)
   - Определение типа отношений: Маитри, Самья, Смешанный, Конфликтный, Критический
   - Подробный анализ по 8 параметрам Кута

3. **Визуализация результатов**
   - Цветовая индикация (зелёный → красный) в зависимости от уровня совместимости
   - Прогресс-бары для каждого параметра совместимости
   - Детальные описания и рекомендации

4. **Дополнительные функции**
   - История расчётов
   - Возможность поделиться результатами
   - Работа как внутри VK, так и в браузере

## 🛠️ Технологический стек

- **Frontend**: React 18 + TypeScript
- **UI Framework**: VKUI 7.0 (дизайн-система ВКонтакте)
- **Роутинг**: VK Mini Apps Router
- **Сборка**: Vite 8.0
- **Интеграция**: VK Bridge для взаимодействия с платформой ВКонтакте
- **Линтинг**: ESLint с конфигурацией TypeScript
- **Разработка**: Hot Module Replacement, TypeScript проверка

## ⚙️ Установка и запуск

### Предварительные требования

- Node.js 18+ и npm/yarn
- Аккаунт разработчика ВКонтакте (для публикации)

### Локальная разработка

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   cd astromeet
   ```

2. **Установка зависимостей**
   ```bash
   npm install
   # или
   yarn install
   ```

3. **Запуск в режиме разработки**
   ```bash
   npm start
   # или
   yarn start
   ```

4. **Открытие в браузере**
   - Приложение будет доступно по адресу: `http://localhost:5173`

### Запуск внутри ВКонтакте

1. **Создание мини-приложения**
   - Перейдите в [Dev Portal ВКонтакте](https://dev.vk.ru/ru)
   - Создайте новое мини-приложение
   - В настройках укажите URL: `http://localhost:5173`

2. **Включение режима разработки**
   - В настройках приложения включите "Режим разработки"
   - Откройте мини-приложение через иконку в интерфейсе VK

### Деплой на хостинг ВКонтакте

1. **Настройка конфигурации**
   - Откройте файл `vk-hosting-config.json`
   - Укажите ID вашего приложения

2. **Запуск деплоя**
   ```bash
   npm run deploy
   # или
   yarn deploy
   ```

3. **Получение ссылки**
   - После успешного деплоя вы получите постоянную ссылку на ваше приложение

## 📊 Система расчёта совместимости

### Основные понятия

- **Накшатра**: 27 лунных созвездий в ведической астрологии
- **Пада**: Четверть накшатры (всего 4 пады на накшатру)
- **Кута**: Система оценки совместимости по 8 параметрам

### Параметры совместимости (8 Кут)

1. **Варна** (Социальная совместимость) - 1 балл
2. **Вашья** (Контроль и доминирование) - 2 балла
3. **Йони** (Сексуальная совместимость) - 4 балла
4. **Гана** (Темперамент) - 6 баллов
5. **Нади** (Здоровье и потомство) - 8 баллов
6. **Маитри** (Дружба) - 5 баллов
7. **Стха-Кута** (Планетарная совместимость) - 3 балла
8. **Бхакут** (Судьба) - 7 баллов

**Максимальный балл**: 36 баллов

### Интерпретация результатов

| Процент | Тип | Описание | Рекомендации |
|---------|-----|----------|--------------|
| 80-100% | Маитри | Идеальная гармония | Отличная основа для брака |
| 50-79% | Самья | Нейтральные отношения | Требуют осознанной работы |
| 30-49% | Смешанный | Сложности в коммуникации | Учитесь принимать различия |
| 10-29% | Конфликтный | Серьёзные противоречия | Требуется много усилий |
| 0-9% | Критический | Крайне неблагоприятно | Консультация астролога |
